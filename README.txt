This is a project README file

This project allows one to take a break every 25 minutes. It is set to go for 4 iterations. You can increment or decrement the minutes or iterations to your desires.

To use open in Python IDLE and simply run. You can replace the weblink with whatever YouTube video or website you would like to have open to aid 
you in taking a break!

Be sure to not take a break for too long!
